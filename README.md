## ğŸ“‹ Project Overview

**WiseGuard AI** is a campaign project for improving community engagement among elderly people through a mobile application. The app focuses on:

1. **Responsible AI Use** - Teaching seniors about AI in simple, jargon-free language
2. **Photo Data Protection** - Explaining the value of their photos and how to protect them
3. **Community Building** - Creating a supportive environment for learning together

### Target Community

- **Primary:** Adults aged 65+
- **Secondary:** Family members helping seniors stay safe online
