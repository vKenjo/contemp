OUTPUT TITLE: Campaign Video on How to Improve Community Engagement Using Mobile

Applications
MAIN OBJECTIVE: The main task of the students is to create a campaign video that can
help improve community engagement using mobile applications. Students MUST choose
a particular mobile application to promote. Alternatively, the students CAN conceptualize a
mobile app that can do such.

SPECIFIC INSTRUCTIONS:
1. This OUTPUT shall be done by GROUPS.
2. The students MUST choose a specific mobile application that can be utilized to
   improve community engagement. If it is possible, the students can conceptualize a
   mobile app that can accomplish the task.
3. The students MUST choose a SPECIFIC target community. This will help
   further contextualizing the campaign video. Moreover, it can help in
   determining the right mobile app for a better community engagement.
4. The students must explain throughout the video how the mobile app can improve
   community engagement in the chosen community. They can be as creative as possible in
   this campaign.
5. For the submission, it will be SUBMITTED thru CANVAS. It will be uploaded first in the
   student’s GDrive. Then, the link will be SHARED in the submission link that will be given.
6. The video will be presented in class (The rubrics for presentation will be given later). The date/s
   of presentation will be determined on a later date.
7. This will be graded based on the rubric below:
